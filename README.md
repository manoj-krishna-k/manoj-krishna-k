**Team Lead | 5-Member Team**

</div>

> A real-time food pre-ordering system that lets hotels prepare meals *before* a customer even arrives — powered by live GPS tracking.

```
📱 Customer App  ←→  🔥 Firebase Realtime DB  ←→  🏨 Hotel Management App
        ↕                                                      ↕
   Live GPS Tracking                               Kitchen Prep Dashboard
        ↕                                                      ↕
   🌐 Spring Boot REST APIs  ←→  Order Lifecycle Management
```

| Feature | Description |
|---|---|
| 📍 **Live GPS Tracking** | Real-time customer location tracked & pushed to hotel dashboard |
| 🍴 **Dual App Architecture** | Separate Flutter apps for Customers & Hotel Staff |
| 🛎️ **Order Lifecycle** | Placement → Preparation → Ready-for-Pickup via Spring Boot APIs |
| 🪑 **Table & Menu Management** | Real-time availability and item updates via Firebase |

### 🧬 SLE Prediction System — ML Diagnostic Engine

</div>

> An ML-powered tool helping medical professionals detect Systemic Lupus Erythematosus (SLE) early through multi-parametric clinical data analysis.

```
Clinical Markers Input (11 Parameters)
├── Anti-dsDNA, C3, C4, CRP, ESR
├── Proteinuria, Platelets, Haemoglobin
└── WBC, Lymphocytes, ANA Titre
         ↓
  RandomForestClassifier (Trained on Synthetic Data)
         ↓
  Diagnostic Prediction → FastAPI Backend → ReactJS Frontend
```
